    \section{IntroducciÃ³n}


    El objetivo de este trabajo es la implementaciÃ³n de un programa para el
ajuste no lineal por mÃ­nimos cuadrados mediante minimizaciÃ³n del
\(\chi^2\) por el mÃ©todo de Newton-Raphson. Este programa se va a
utiizar para ajustar unos datos experimentales a una funciÃ³n
\(y=\exp(ax)+bx^2\) para estimar los parÃ¡metros \(a\) y \(b\) Ã³ptimos.
Adicionalmente, se van a estimar las incertumbres del ajuste a partir de
la matriz de informaciÃ³n de Fisher y simulaciones Monte Carlo.


    \section{Fundamento TeÃ³rico}


    Sea un conjunto de \(N\) medidas \(y_i\) (con errores \(\sigma_i\)) para
valores \(x_i\) a los que se les quiere ajustar una funciÃ³n
\(f(x_i;a_j)\) que predice los valores de \(y_i\), donde \(a_j\) con
\(j=1,..,p\) son parÃ¡metros desconocidos. Los valores Ã³ptimos de los
parÃ¡metros \(a_j\), de acuerdo con el mÃ©todo de mÃ­nimos cuadrados, para
el ajuste se pueden estimar minimizando el \(\chi^2\):

\[\chi^2 = \sum_{i=1}^{N} \frac{[y_i -f(x_i;a_j)]^2}{\sigma_i^2} \]

que sigue aproximadamente una distribuciÃ³n \(\chi^2\) con \(N-p\) grados
de libertad, hecho que puede utilizarse para evaluar la bondad del
ajuste como la probabilidad del \(\chi^2\) real sea mayor que el
\(\chi2\) observado.

La minimizaciÃ³n para funciones sencillas se puede realizar de forma
analÃ­tica, sin embargo, en general se requiere la minimizaciÃ³n numÃ©rica
del \(\chi^2\) para la obtenciÃ³n de los parÃ¡metros. Existen multitud de
mÃ©todos de minimizaciÃ³n numÃ©rica, aunque el mÃ©todo que se va utilizar en
este trabajo es el Newton-Raphson (NR) que sirve para buscar el cero de
la funciÃ³n, por lo que se puede utilizar para buscar el cero de del
gradiente de \(\chi^2\) con respecto a los parÃ¡metros.

SupÃ³ngase una funciÃ³n \(f(x_1,...,x_n)\) que depende de varios
parametros \(\bf{x}\), se puede desarrollar en serie en primer orden
alrededor de un valor inicial \(\bf{x^{(0)}}\) segÃºn:

\[\nabla f({\bf{x}}) = \nabla f ({\bf{x}}^{(0)}) 
+ \bf{H}({\bf{x}}- {\bf{x}}^{(0)}) ~+~ ... \]

en donde \(\bf{x}\) es el vector de parÃ¡metros, \(\nabla f({\bf{x}})\)
es el vector gradiente y \(\bf{H}\) es la matriz hessiana. Si se impone
que \(\nabla f ({\bf{x}}^{(0)}) = 0\), es decir que hay un extremo en
\({\bf{x}}^{(0)}\), se obtiene el siguiente sistema de ecuaciones:

\[{\bf{x}}^{(i+1)} = {\bf{x}}^{(i)} - {\bf{H}^{-1}}\nabla f ({\bf{x}}^{(i)})\]

que se puede resolver de forma iterativa para obtener los valores de
\({\bf{x}}\) que corresponde a un extremo. Este mÃ©todo puede aplicar
para encontrar el mÃ­nimo de la funciÃ³n \(\chi^2\) a partir de las
derivadas primeras y segundas de la misma con respecto a los parÃ¡metros
\(a_j\).

Una vez que se han estimado los valores Ã³ptimos para los parÃ¡metros
\(\hat{a}_j\), la matrÃ­z de covarianza \(V\) puede obtenerse utlizando
la matrÃ­z de informaciÃ³n de Fisher, que se puede relacionar con matrÃ­z
hessiana para los mejores valores de los parÃ¡metros obtenidos. La
inversa de la matriz de covarianza para el \(\chi^2\) se obtiene como:

\[(V^{-1})_{ij} = \left. \frac{1}{2} \frac{\partial^2 \chi^2 }{\partial a_i \partial a_j } \right|_{a_j = \hat{a}_j}\]

en donde el error de los parÃ¡metros estimados se puede obtener como la
raÃ­z cuadrada de los elementos de la diagonal de la matrÃ­z de covarianza
\(V\).

Adicionalmente, en este trabajo se va a estimar el sesgo y el error en
los parÃ¡metros utilizando simulaciones Monte Carlo. En las simulaciones
se va suponer que los errores son gaussianos y se van a realizar un
total de \(N\) simulaciones. Para cada simulaciÃ³n \(s\), el valor de
\(y_i^s\) puede obtenerse como:

\[y_i^s = f(x_i;a_j) + n_i^s ~{\rm{con}}~ s=1,...,N\]

en donde \(n_i^s\) es el error simulado para cada punto de datos que es
un nÃºmero aleatorio que sigue una distribuciÃ³n gaussiana con una
desviaciÃ³n estandar \(\sigma_i\) y \(a_j\) son los parÃ¡metros del
modelo. El valor de los parÃ¡metros de ajuste \(\hat{a}_j\) puede volver
a ser estimado para cada simulaciÃ³n, a partir de los cuales se puede
estimar el sesgo \(b_j\) y el error \(\sigma({\hat{a}_j})\):

\[b_j = \left< \hat{a}_j^s \right> - \hat{a}_j^{data} \]

\[\sigma({\hat{a}_j}) = \sqrt{\left< \hat{a}_j^2 \right> - \left< \hat{a}_j \right>^2 }\]

en donde los parÃ©ntesis angulares indican el promedio sobre todas las
simulaciones, \(\hat{a}_j^2\) es la estimacion del parÃ¡metro para cada
simulaciÃ³n y \(\hat{a}_j^{data}\) es la estimaciÃ³n de los parÃ¡metros de
los datos. Si el sesgo de un estimador es cero, este estan insesgado. La
estimaciÃ³n final del los parÃ¡metros se puede obtener segÃºn:

\[a^{final}_i = (\hat{a}_i - b_i) \pm \sigma(\hat{a}_i)\]

que se puede comparar con la obtenida a partir de informaciÃ³n de Fisher.
